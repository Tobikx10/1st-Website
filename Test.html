Pronajměte si vlastní Minecraft server

Nabízíme Vám vysoce kvalitní servery již od 129,- Kč měsíčně za 2GB ram s neomezenými sloty nebo 135,- Kč měsíčně za 5 slotů a 3GB ram. Za tuto nízkou sumu získáte Minecraft server a k němu velké množství nadstandartních služeb zdarma. Mezi tyto služby patří komunikační server (TeamSpeak 3 / Mumble), FTP přístup k souborům serveru, Control Panel - uživatelsky přívětivá webová administrace (zapnutí/vypnutí/restart serveru, statistiky, editace souborů serveru, nahrávání záloh, reinstalace serveru), MySQL databázi, pravidelné zálohování (Minecraft server co 2 hodiny, MySQL 1x / den). Samozřejmostí je také kvalitní technická podpora a pomoc na našem discordu, která Vám pomůže s instalací a nastavením pluginů.

Při pronájmu herního serveru u nás získáte také možnost použít unikátní obrázkový status Vašeho serveru. Jako první CZ/SK hosting nabízející Minecraft servery máme SMS platební systém. Tuto jednoduchou platební metodu můžete použít jako příspěvkový/VIP platební kanál.

Hrajete na serveru vytvořeném doma nebo jinde a nechcete ztratit vytvořený svět?

Toto není problém, svůj svět můžete nahrát na náš server pomocí FTP. Pokud máte pomalý internet, nevěšte hlavu. Vypalte si zálohu Vašeho serveru na CD/DVD, zašlete je nám a my svět nahrajeme.

Proč si Minecraft server pronajmout, když můžete hrát přes Hamachi?

  • U nás hostované servery jsou dostupné 24 hodin denně, 7 dní v týdnu.
